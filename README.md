# PROG2
This repo will store all of my  projects which were created during SS 24/25 in uni.

## Assignment 0 
The goal of the assignment is to write a simple C program that prints the prime factors of a given number. This was a testing assignment so we could understand how the system works.

## Assignment 1 
The aim of the assignment is to write a C program that displays a formatted table with calculated values of artificial neuron activation functions. In the assignment we practiced working with mathematical operations, decimal numbers, functions, loops and formatting the output text. This assignment is part of a larger project to develop software for recognising handwritten digits in an image using a neural network.

## Assignment 2
The aim of the assignment is to write a program to classify images of handwritten digits from the MNIST database using a neural network. The task of the assignment is to implement the basic operations of the neural network, which we will eventually use to classify the image. Finally, we evaluate the success of the neural network in classifying multiple images.

## Assignment 3
The goal of the assignment is to write a console word processor that will use commands to process the lines of text that are loaded.


## Assignment 4
The goal of the assignment is to write a console application that will simulate a battle between two armies on the battlefield. The armies will consist of combat units. Each unit can have different items in its inventory (from the item database). The user will configure both armies before the duel starts. The duel is conducted in rounds, in which the units engage each other using their items in the inventory. The duel can end with either army winning or drawing

## Assignment 5
The aim of the assignment is to write a console application, which will be an extension of assignment 4. The extension will consist of adding text files from which objects and combat units are loaded. The objects will be loaded from a JSON file. Combat units in the army will be loaded from TXT files. During loading, the focus will be on detecting various error situations.
